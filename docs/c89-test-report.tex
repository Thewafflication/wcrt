\begin{abstract}
This report specifies the compile-only API presence tests for the WCRT C89
hosted-library milestone. Test cases TC-0001 through TC-0015 map one-to-one to
requirements REQ-0001 through REQ-0015 and the corresponding public C89
headers. Functional, ABI, boundary, and platform behavior are deferred to later
test cases.
\end{abstract}
