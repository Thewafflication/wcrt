\begin{abstract}
This master report specifies the WCRT C89 tests and includes available execution
results and evidence. Test cases TC-0001 through TC-0016 map one-to-one to
requirements REQ-0001 through REQ-0016. Every test is executed for WCRT even
when its requirement is not yet implemented. Microsoft UCRT API-presence runs
provide a comparison baseline; project source lint is not applicable to UCRT.
\end{abstract}
